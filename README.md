# Just_Chatting

> Client, Server 모두 Typescript로 이루어져 있습니다.

## Init

```shell
> yarn install
```

## <br />

## Client

### Run

```shell
> yarn client:dev
```

### Stacks

- NextJS 13 (Framework)
- Emotion (Style)
- Jotai (Store)

## <br />

## Server

### Run

```shell
> yarn server:dev
```

### Stacks

- NestJS (Framework)
- MySQL (Database)
- TypeORM (ORM)
- JWT (Auth)
